## 🔨 Tools

- [LangChain](https://python.langchain.com/docs/introduction/)
- [Streamlit](https://docs.streamlit.io/)

## ⚡️ Instructions

- Clone the repo
- Create and activate virtual environment
- Install dependencies

  ```sh
  pip install -r requirements.txt
  ```
- Run the application

  ```sh
  streamlit run app.py
  ```
